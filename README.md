## OLED视频格式的要求
- 如果你想要oled播放视频，那你就得需要flash模块！(64kb flash实在难以发挥)
- #### Flash格式:
1. 在BasicAddr之后需要一个int来指示pointer的总数目
2. 紧接着int后面是pointer的数组(uint32_t)
3. pointer之后是你的视频data

- 注意pointer是指向某一帧的地址
- pointer的目的是为了能够轻松实现快进、快退功能
- 通过更改BasicAddr就可以做到换视频
- _建议使用ESP32S3来烧录flash_
